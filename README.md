# AI Content API

Небольшой pet-проект, который я сделал, чтобы разобраться, как вообще строятся backend API и как можно работать с текстом.

Идея простая: есть сервис, который принимает текст и делает с ним разные штуки — исправляет ,упрощает или генерирует что-то новое.

## Что умеет

* `/correct` — исправляет простой текст (пока на правилах, без AI, тут думаю разбираться как будет время, это на днях) 
* `/simplify` — упрощает фразы (типа делает их более разговорными, а почему бы и да)))
* `/generate` — генерирует текст по prompt (сейчас заглушка, дальше планирую подключить AI)

## Технологии

* Python
* FastAPI
* Pydantic
* Uvicorn

## Как запустить

1. Клонировать репозиторий:

```
git clone https://github.com/AndreiPabiarzhyn/ai-content-api.git
```

2. Перейти в папку проекта:

```
cd ai-content-api
```

3. Создать виртуальное окружение:

```
python -m venv venv
venv\Scripts\activate
```

4. Установить зависимости:

```
pip install -r requirements.txt
```

5. Запустить сервер:

```
uvicorn app.main:app --reload
```

После этого можно открыть:

* http://127.0.0.1:8000/docs — там удобно тестировать API. Это блин прям магия FastApi- круто
